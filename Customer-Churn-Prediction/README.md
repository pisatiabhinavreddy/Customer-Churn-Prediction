# 📊 Customer Churn Prediction (Telco Dataset)

## 📌 Project Overview
This project predicts customer churn using machine learning techniques on the Telco Customer Churn dataset. It helps identify customers who are likely to leave a service.

---

## 🎯 Objective
To build a classification model that predicts whether a customer will churn (`1`) or not (`0`).

---

## 📂 Dataset
- Dataset: Telco Customer Churn Dataset  
- File: WA_Fn-UseC_-Telco-Customer-Churn.csv  

### Features:
- Customer demographics  
- Service subscriptions  
- Monthly charges  
- Total charges  
- Contract details  

### Target:
- Churn (Yes/No → Converted to numeric)

---
